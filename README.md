# Coverage Counters

## Cyclomatic Complexity

1. JaCoCo рассчитывает цикломатическую сложность метода с помощью следующего эквивалентного уравнения:
 
 ``v(G) = B - D + 1``
 
где  (B)- количество ветвей, а (D) - количество точек принятия решения.
